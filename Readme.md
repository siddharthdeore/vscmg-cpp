## Systems
- Rigid Body
- Variable Speed Control Moment Gyroscope

## Build
```
mkdir build && cd build
make
```
## Examples
```
# Rigid Body 
./TestRigidBody

# Variable Speed Control Moment Gyroscope
./TestVSCMG
```

## Requirements
- C++ 14
- [Boost](https://www.boost.org/)
- [Eigen3](https://eigen.tuxfamily.org/index.php?title=Main_Page)
